# Liquid

Liquid is a C++ standalone unidirectional pathtracer.

Features:

- [x] Uniform sampler
- [x] CPU multithreading (local)
- [x] BVH acceleration (objects [top])
- [ ] BVH acceleration (polygons [bottom])
- [ ] SIMD support
- [ ] Importance sampling
- [x] IBL with HDR maps
- [ ] Disney principled shader

Materials:

- [x] Lambertian
- [x] Diffuse Light
- [x] Metal
- [x] Glass
- [ ] Volumes
